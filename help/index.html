Get Started with HoAh AI

HoAh uses powerful AI models to polish and translate your voice. To do this, you need a "Key" from an AI provider. Don't worry, it's easy and often free!

1. What is an API Key?
Think of an API Key as a digital VIP Pass. It allows HoAh to send your text to a super-smart AI brain (like ChatGPT) for processing and get the result back. You are in full control of this key.
2. Choose a Provider

3. Alternative: Run AI Locally with Ollama

Don't want to use cloud APIs? Ollama lets you run AI models entirely on your Mac — no API key, no internet, no cost. Perfect for privacy-focused users.

🦙

Ollama (Local)

Run AI models locally on your Mac. 100% free, 100% private. No API key needed.

Setup in 3 steps:

  1. Install Ollama from ollama.com
  2. Open Terminal and run: ollama pull qwen2.5:1.5b
  3. In HoAh, add an AI Configuration → select Ollama (Local) as the provider

Recommended models:

qwen2.5:1.5b — Fast, lightweight (default)
llama3.2:3b — Good balance of speed and quality
gemma2:2b — Google's compact model
phi3:3.8b — Microsoft's efficient model

Tip: Run ollama list to see installed models. If HoAh can't connect, make sure Ollama is running (ollama serve).
